# AWS Serverless Job Application Portal

## Project Overview

A serverless web application that allows candidates to submit job applications through a web form. The application uses AWS API Gateway and AWS Lambda to process requests and CloudWatch for monitoring.

## Architecture

Frontend (HTML/CSS/JavaScript)
        ↓
AWS API Gateway
        ↓
AWS Lambda
        ↓
Amazon CloudWatch

## Features

- Responsive Job Application Form
- Client-side Form Validation
- AWS API Gateway Integration
- AWS Lambda Backend Processing
- CloudWatch Logging and Monitoring
- Serverless Architecture
